# e-Accounting 시스템 기본 기능

## 로그인/대시보드

### e-Accounting 시스템 접속

1. 포털 메인화면 하단에서 **지원** 클릭 → 'e-Accounting' 클릭
2. 웹브라우저(Microsoft Edge에 최적화)에서 직접 접속: [https://eacct.gsretail.com](https://eacct.gsretail.com)

**활용 용도 및 목적:**
- e-Accounting 접속
- 로그인, 비밀번호 변경/초기화

---

### 로그인/비밀번호 변경

1. 사용자 본인의 포탈ID(e-Mail 주소에서 `@gsretail.com` 제외) 입력
2. 비밀번호 입력
3. **로그인** 클릭
4. 회사 보안규정에 따라 비밀번호 변경 또는 초기화 필요시 클릭
5. 인사정보에 등록된 사용자의 휴대전화에 발송된 인증번호 입력
6. 사용할 패스워드 입력
7. **수정버튼** 클릭

**특이사항:**
- e-Accounting에 SSO(Single Sign On)는 적용되지 않음.
*SSO(Single Sign On) : 한 번의 로그인(인증)으로 여러 서비스(시스템)을 이용할 수
있도록 하는 구조
---

### 메인 화면

1. **메뉴 List:** 전표조회/작성, 리포트, 마스터 관리 등
2. **Dashboard:** 공지사항 등
3. **요약 현황:** 법인카드/세금계산서 미처리건, 상신대기/결재진행 중 등 결재관련 요약 정보 제공
4. **접속정보 등:** 운영자 현황, 로그인 정보
5. **새탭으로 열기:** 다중창 필요 시 클릭하여 사용

**특이사항:**
- 사용자별 권한이 다르므로, 업무상 필요한 권한은 GRIT을 통해 운영자에게 신청 필요
- 좌측 'Menu Search' 기능을 통해 원하는 메뉴를 바로 검색가능

---

### 상단 요약 현황

1. **법인카드 미작성:** 사용자의 법인카드 사용내역(위임 받은 내역 포함) 중 전표 미처리건수, 클릭 시 전표작성 화면으로 이동
2. **세금계산서 미작성:** 사용자에게 지정된 전자(세금)계산서 중 미처리 건 수, 클릭 시 (세금)계산서 전표작성 화면으로 이동
3. **상신대기:** 전표 저장 후, 상신(결재요청)하지 않은 건수
4. **상신:** 결재 요청 후, 1차 결재자가 결재하지 않은 상태
5. **결재진행중:** 1차 결재자가 결재하였고, 최종 승인되지 않은 상태
6. **결재수신:** 내가 결재해야 하는 전표의 건수
7. **금일결재반려:** 내가 상신한 전표 중, 오늘 반려(부결) 처리된 건수

**특이사항:**
- 전표 상태가 '상신'일 경우 전표 작성자가 전표 회수가능
- 전표 상태가 '결재진행중'인 경우 전표 회수 불가능, 다음 결재자가 반려해야만 취소 가능

---

## 공통 기능

### 전표 작성 화면 구성

1. **제목 영역:** 해당 메뉴의 제목을 표시
2. **입력(전표 헤더) 영역:** 회계반영일(전기일), 증빙유형 등 전표의 기본정보 입력
3. **그리드(Grid) 영역:** 차/대변 계정과목, 귀속부서, 금액 등 재무제표 반영 항목
4. **증빙/기안문(품의) 연결:** 전표 결재에 필요한 증빙 파일(예: 인보이스, 명세서, 영수증, 세금계산서, 엑셀파일) 및 전자결재 내 품의서 연동
5. **결재선 정보:** 로직에 따라 결재선 호출, 필요에 따라 결재선 변경 가능하나, 가급적 호출된 결재선 준수 권장
6. **결재요청의견:** 전표의 내용, 특이사항 등 결재자가 인지해야할 정보 입력 (필수 아님)
7. **저장/결재요청:** 저장은 입력된 값의 임시저장, 결재요청은 클릭 후 즉시 상신

### 파일 첨부

#### 전표 작성 메뉴 내 화면에서 파일 첨부

1. 입력화면에서 **첨부 버튼** 클릭
2. 파일 첨부 팝업이 호출되며, **Browse 버튼** 클릭 후, 파일을 첨부하고 **적용 버튼** 클릭
3. 또는, 파일이 들어있는 윈도우 창에서 파일 선택 후, Drag로 팝업에 복사 후 **적용 버튼** 클릭


---

### 기안문(품의서) 연결

1. **일반적인 형태:** 입력화면에서 **GW검색 버튼** 클릭
2. **사번 또는 성명으로 검색**
3. **품의일자 입력**
4. **품의서 제목 입력** 후 조회
5. 조회된 리스트에서 연결할 품의서 선택
6. **적용 버튼** 클릭, 연결 완료

**특이사항:**
- 품의서(전자결재) 상, **전표작성자**로 지정되어야 `사번 또는 성명으로 검색`이 가능
- 전표작성자 지정은 품의서 기안자가 '포탈>전자결재>결재한문서'에서 해당 품의 클릭 후, 상단의 **'전표작성자등록' 버튼**을 눌러 지정 가능

---

### (기본) 입력/선택 방법

1. **일자지정(회계반영일 등):** 날짜(Text) 직접 입력, 또는 우측 달력 아이콘 클릭 후 선택
2. **콤보박스(증빙유형 등):** 하단으로 향하는 화살표 모양 아이콘 클릭 후 선택
3. **코드/명칭 입력:** 코드(부서, 회계계정과목 등)와 명칭(경영회계팀, 노무관리비 등)의 직접 입력, 또는 돋보기 아이콘으로 팝업 창 호출

---

### 그리드 사용 방법



[[[CHUNK]]]

#### 그리드 사용 방법 – 필터 화면 설명

1. **(일반적으로) 필터 값 입력행:** 그리드의 첫 번째 줄
2. 필터를 적용할 열을 클릭하면 **필터 설정** 가능
3. (예시) **가맹점명** 열에 "GS"를 넣고 엔터: "GS"를 포함하는 항목만 리스트에 조회
4. (예시) "총 금액"열에 100,000 입력후 X 표시를 눌러 > 기호로 변경: 100,000원 미만인 항목만 리스트에 조회

#### 그리드 사용 방법 – 페이지 단위로 보기

- 첫 페이지로 이동
- 페이지 간 이동 가능
- 페이지 단위로 나눠서 보기
- 필요 시, 체크박스를 해제하여 전체 리스트 보기 가능

#### 그리드 사용 방법 – 엑셀 붙여 넣기

1. 엑셀에서 복사할 영역 선택하여 **Ctrl + C**
2. 붙여 넣을 그리드에 셀 선택하면, 붉은색 테두리로 표시
3. ESC키를 한 번 눌러서, 파란색 테두리로 변경
4. **Ctrl + V**로 붙여 넣기 완료

**특이사항: ESC키를 누르지 않고 바로 붙여넣을 경우, 첫 행의 데이터가 누락될 수 있음**

#### 그리드 사용 방법 – (필터 행을 활용한) 일괄 처리
**동일 값 일괄 입력**
1. 값을 입력할 라인 선택
2. **필터행** 에 값을 입력 후 엔터
3. 동일 값이 선택한 라인에 일괄 입력됨

**증빙첨부 및 기안문(품의서)연결 일괄 처리**
1. 같은 파일을 첨부할 라인을 선택
2. **필터행**을 더블 클릭
3. **파일 첨부**하고, **등록**버튼 클릭
4. 동일 파일이 일괄 첨부되며 아이콘이 변경됨

---

### 결재라인 생성/변경

1. (전표 작성 등)화면에서 계정과목/금액 등 입력 후, **결재정보 가져오기** 버튼 클릭
2. 업무유형/회계계정과목별로 결재라인 자동 지정
3. 필요 시, 결재라인에 추가할 인원 선택 후 추가(팝업 화면)
4. 결재라인 편집 가능: 결재유형(결재/합의) 선택, 결재자 삭제, 순서 변경
5. 결재라인 변경시 사유입력 필수

**특이사항: 결재라인 변경 전표는 모니터링 대상임**

---

### 결재라인 추가 팝업 화면 설명

1. **코드(사번) 또는 텍스트(성명) 입력**
   - 유사 검색 가능
2. **검색버튼 클릭**
3. 검색결과 중, 선택 라인을 더블클릭
4. 검색결과에서, 그리드 필터 기능을 활용하여 추가검색 가능
